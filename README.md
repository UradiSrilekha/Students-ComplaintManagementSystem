Student Complaint Management System 🎓
A web-based platform for students to submit, track, and manage their complaints efficiently. Built using HTML, CSS, and JavaScript with a responsive and user-friendly interface.

🚀 Live Demo: https://students-complaint-management-syste-ten.vercel.app/

📖 Overview
The Student Complaint Management System is a modern web application designed to streamline the process of submitting and managing student complaints within an academic institution. Built entirely with HTML, CSS, and JavaScript, this application provides an intuitive interface for students to raise issues, view complaint status, and receive updates. The system ensures a clean, responsive, and accessible design suitable for both desktop and mobile users.

✨ Features
Feature	Description
📝 Complaint Submission	Students can submit complaints with title, description, and category
👁 Complaint Tracking	View submitted complaints and their current status
🔍 Search & Filter	Easily search and filter complaints by status or category
📱 Responsive Design	Fully optimized for desktop, tablet, and mobile devices
🎨 Clean UI	Modern and intuitive interface with smooth interactions
🛠️ No Backend Required	Built purely with frontend technologies for simplicity and quick deployment

🚀 Getting Started
Prerequisites
Ensure you have the following installed:

Node.js (v14 or higher) (optional, for local testing with live servers)

Any modern web browser

Installation
Clone the repository:


bash
git clone https://github.com/UradiSrilekha/Student-Complaint-Management-System.git
cd Student-Complaint-Management-System
Open the project in your browser:

Simply open index.html in any modern browser.

Or use a live server extension in VS Code for better preview.

Running the App Locally
Option 1: Direct Browser Opening
Navigate to the project folder and open index.html in your browser.

Option 2: Using Live Server (VS Code)
Install the Live Server extension in VS Code.

Open the project folder in VS Code.

Right-click on index.html and select “Open with Live Server”.

Your app will be accessible at http://localhost:5500.
